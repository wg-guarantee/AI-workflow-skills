# 通用业务流程图生成器

根据业务描述自动生成交互式HTML流程图。

## 功能特点

- 📊 自动解析业务描述，识别阶段、角色、步骤
- 🎨 生成交互式HTML流程图
- 📱 响应式设计，支持多端访问
- 🔄 支持阶段切换查看
- 🎯 支持多角色协作展示

## 使用方法

### 在 Claude Code 中使用

直接描述你的业务流程，例如：

```
帮我生成一个电商订单处理流程图，包含以下阶段：
1. 用户下单 - 用户选择商品下单
2. 订单确认 - 系统确认订单，商家确认
3. 支付处理 - 用户支付，支付平台处理
4. 发货 - 商家发货，物流配送
5. 收货确认 - 用户确认收货
```

### 作为独立工具使用

```bash
cd ~/.claude/skills/flowchart-generator
tsx skill.ts "你的业务描述"
```

## 输出格式

生成独立的HTML文件，包含：
- 完整的流程图布局
- 阶段切换功能
- 角色列展示
- 操作类型标识（线上/线下/API）

## 示例

### 输入
```
# 用户注册流程

阶段1：填写信息
1. 用户：输入用户名、密码、邮箱

阶段2：验证信息
1. 系统：验证邮箱格式，检查用户名重复

阶段3：发送验证码
1. 系统：生成验证码，发送邮件
2. 用户：接收邮件，输入验证码

阶段4：完成注册
1. 系统：验证码校验，创建账户
2. 用户：注册成功，自动登录
```

### 输出
生成可交互的HTML流程图，展示4个阶段的完整流程。
