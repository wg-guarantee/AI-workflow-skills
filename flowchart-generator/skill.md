# 通用业务流程图生成器

## 功能

根据用户的业务描述，自动生成交互式HTML业务流程图，支持：
- 多阶段流程展示
- 多角色协作
- 可编辑流程节点
- 可拖拽调整流程箭头
- 响应式布局

## 使用方法

输入你的业务流程描述，例如：
```
我需要做一个电商订单处理流程图，包含以下阶段：
1. 用户下单 - 用户选择商品下单
2. 订单确认 - 系统确认订单，商家确认
3. 支付处理 - 用户支付，支付平台处理
4. 发货 - 商家发货，物流配送
5. 收货确认 - 用户确认收货
6. 售后（可选）- 处理退换货
```

## 输出

生成一个独立的HTML文件，包含：
- 完整的流程图布局
- 阶段切换功能
- 箭头连接和编辑
- 本地存储支持
